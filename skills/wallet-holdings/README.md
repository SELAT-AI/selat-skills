# wallet-holdings

**Single-rail** · DIRECT (Circle nanopayment)

Multi-chain token holdings for a wallet via Alchemy, paid directly over Gateway-batched.

## Params

| Param | Required | Notes |
|---|---|---|
| `address` | yes | Wallet address, `0x...` |

## Run

```bash
selat skill install wallet-holdings
selat skill run wallet-holdings --address 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045
```

## Cost

≈ $0.005 per run.
